# 快速开始

## 1. 环境要求

- Python 3.10+
- Git（已安装并在 PATH 中）
- 能读项目文件并执行终端的 **AI 助手**（Cursor、Claude Code、Windsurf 等，用于按 `SKILL.md` 生成日报）

## 2. 安装

```bash
cd D:/javaPro/daily-work-digest
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 3. 配置

```bash
copy config.example.yaml config.yaml
```

编辑 `config.yaml`，在 `git.repos` 中填入你要扫描的仓库绝对路径，例如：

```yaml
git:
  repos:
    - D:/javaPro/your-project
    # 勿将本工具目录 daily-work-digest 配入 repos
  author_email: your@company.com   # 可选，只采集自己的提交
```

可选：启用飞书日程（CalDAV）：

```yaml
feishu:
  enabled: true
  caldav:
    server: https://caldav.feishu.cn
    username: "飞书 CalDAV 用户名"
    password: "飞书 CalDAV 专用密码"
```

详见 [飞书日程采集-用户指南.md](./飞书日程采集-用户指南.md)。

## 4. 采集今日工作记录

```bash
python scripts/collect_all.py --date today
```

成功后会产生：

- `data/raw/2026-06-16/git.json` — Git 原始采集
- `data/digest/2026-06-16.json` — 归并后的事件列表

## 5. 手动补记（可选）

创建 `data/manual/2026-06-16.md`，参考 [手动补记示例.md](./手动补记示例.md)。

然后重新归并：

```bash
python scripts/merge_daily.py --date today
```

## 6. 用 AI 助手生成日报

打开本项目，让 AI 助手按 `SKILL.md` 成文。示例：

```
生成今天的工作日报
```

Cursor 用户也可：

```
@SKILL.md 生成今天的工作日报
```

Claude Code 用户可在项目目录说：

```
按 SKILL.md 生成今天的工作日报
```

Agent 会读取 digest 并按模板成文。

## 7. 生成周报

先确保本周各天已有 daily digest，然后：

```bash
python scripts/merge_daily.py --date today --week
```

在 AI 助手对话中说（或 Cursor 使用 `@SKILL.md`）：

```
根据本周 digest 生成周报
```

## 8. 定时采集（可选）

使用 Windows **任务计划程序**，每天 18:00 执行：

```
D:\javaPro\daily-work-digest\.venv\Scripts\python.exe
D:\javaPro\daily-work-digest\scripts\collect_all.py --date today
```

## 常见问题

| 问题 | 处理 |
|------|------|
| `git.repos 为空` | 检查 config.yaml 是否已配置仓库路径 |
| 某仓库 `不是 Git 仓库` | 确认路径正确且含 `.git` |
| digest 为空 | 当天可能无提交；用手动补记补充 |
| 找不到 config.yaml | 从 config.example.yaml 复制 |
| `No time zone found with key Asia/Shanghai` | 执行 `pip install tzdata` |

更多设计细节见 [技术文档.md](./技术文档.md)。
