# 语雀文档采集 — 用户指南

通过语雀 API 拉取指定日期「**本人编辑过**」的文档 **L1 元数据**（标题、链接、更新时间等），**不拉取全文**。

支持两种认证方式，**任选其一**（在 `config.yaml` 中配置）：

| 方式 | 适用场景 |
|------|----------|
| **Token**（默认） | 有超级会员，可在 [Token 设置页](https://www.yuque.com/settings/tokens) 创建 |
| **Cookie** | 无超级会员，从浏览器复制登录 Cookie |

---

## 普通用户怎么用

对 AI 助手说：

```text
帮我配置语雀文档采集
```

你只需要：

1. **Token 或 Cookie**（复制到 `config.yaml`）
2. **`enabled: true`**

**不需要手动填 `repos`** — 留空时会自动扫描你账号下的知识库。

---

## 最简配置

**Cookie 模式（无超级会员）：**

```yaml
yuque:
  docs:
    enabled: true
    auth_mode: cookie
    cookie: "从浏览器复制的 Cookie"
    repos: []          # 留空即可，自动发现知识库
```

**Token 模式：**

```yaml
yuque:
  docs:
    enabled: true
    auth_mode: token
    token: "你的Token"
    repos: []
```

## 获取凭证

### Token

1. 打开 [语雀 → 设置 → Token](https://www.yuque.com/settings/tokens)
2. 新建 Token 并复制

> 若页面提示需超级会员，改用 Cookie 模式。

### Cookie

1. 登录 [语雀](https://www.yuque.com)
2. F12 → **Network** → 刷新页面
3. 复制任意请求的 **Cookie:** 完整字符串（含 `_yuque_session`、`yuque_ctoken`）

也可只粘贴 `_yuque_session` 的值，程序会自动补全。

> Cookie 等于登录凭证，**切勿泄露或提交 Git**。`config.yaml` 已在 `.gitignore`。

Cookie 会过期（数天～数周），401 时重新复制即可。

## repos（可选，默认不用配）

| 配置 | 行为 |
|------|------|
| `repos: []`（默认） | **自动扫描**账号下全部知识库 |
| `repos: [login/slug, ...]` | 只扫描指定知识库（更快、范围更窄） |
| `auto_repos: false` | 关闭自动发现，此时必须填 `repos` |

查看你的知识库列表（调试用）：

```bash
python scripts/debug_yuque_docs.py --list-repos
```

python scripts/collect_all.py --date today --sources yuque_docs

## 配置说明

| 字段 | 说明 | 默认 |
|------|------|------|
| `auth_mode` | `token` 或 `cookie` | `token` |
| `token` / `cookie` | 对应模式的凭证 | `""` |
| `repos` | 知识库白名单；**留空 = 自动发现** | `[]` |
| `auto_repos` | 是否自动发现知识库 | `true` |
| `max_auto_repos` | 自动发现时最多扫描几个 | `50` |
| `only_my_edits` | 仅保留本人编辑的文档 | `true` |
| `use_content_updated_at` | 按正文更新时间筛日期 | `true` |
| `also_use_updated_at` | 同时认元数据 `updated_at` | `true` |
| `page_limit` | 单页文档数（最大 100） | `100` |
| `max_pages` | 每个知识库最多翻页次数 | `20` |

---

## 常见问题

| 现象 | 处理 |
|------|------|
| 401（Token） | Token 无效，重新创建 |
| 401（Cookie） | Cookie 过期，从浏览器重新复制 |
| 0 篇文档 | 当天未编辑；或 `only_my_edits` 过滤 |
| 扫描太慢 | 用 `repos` 白名单缩小范围 |
| Token 页需超级会员 | 改用 `auth_mode: cookie` |

## 采集内容（L1）

| 有 | 没有 |
|----|------|
| 标题、链接、编辑时间 | 正文全文 |
| 知识库、编辑者 | |

输出：`data/raw/{date}/yuque_docs.json` → digest 中 `source: yuque`, `type: document`。
